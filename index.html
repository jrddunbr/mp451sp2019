dnssd

dnssd (the domain name system server daemon) is a piece of software using ed25519 cryptography to dynamically assign domain names to different computers. Not finished, but a protocol RFC has been created and a partial implementation as well. Written in Python

Electrical Age

I've been contributing to Electrical Age development over the last year with Graham Northup, and intend to continue development through next semester and beyond. Administrating the most popular fork of a popular Minecraft mod is a great way to learn about how to handle an effective Git workflow, as well as working with other people.

See more at my Electrical Age GitHub fork

eln-matrix-viz

eln matrix vis is a compiler from a special output format from an Electrical Age debugging command to dot files. These dot files can then be passed into GraphViz to create neat images of connected node graphs

Outlet

A Git repository full of cancerous code. I made some contributions, and also learned how Git hooks work. All of the "Build failing" in the makefile from my Git user are because I had a pre-commit hook to add that text, so every time I commited, it added that.

Laundry

A python script designed to fetch json from the LaundryView website and tell me what washers and dryers are available in all of the buildings, and how much longer they have to completion, so that I can see how my laundry is doing.

wtherm

IoT hacking of a WiFi enabled meat thermometer. We discovered how to parse some of the packets well enough to get the temperature from the probe, and set the temperature to beep at. Apparently, you can do many other horrible things like set the currently read temperature by fuzzing the packets.
